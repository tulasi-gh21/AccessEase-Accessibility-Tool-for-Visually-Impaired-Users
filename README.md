♿ AccessEase – Accessibility Tool for Visually Impaired Users

AccessEase is a frontend web accessibility project designed to improve website usability for visually impaired and low-vision users. The application provides accessibility features such as text zoom controls, dark mode, high-contrast mode, and text-to-speech voice readout to make web content easier to read and interact with.

The project was built using HTML, CSS, and JavaScript with a focus on accessibility and user-friendly interface design.

🚨 Problem Statement

Many websites are difficult for visually impaired users to use because:

Text size is too small
Color contrast is poor
Content cannot be read aloud
Websites are not accessibility-friendly

AccessEase was created to solve these problems by providing simple accessibility controls directly on the webpage.

✨ Features
Increase and decrease text size
Dark mode support
High contrast reading mode
Text-to-speech voice readout
Stop voice reading functionality
Responsive and beginner-friendly UI

🔊 How the Voice Readout Works

The project uses the browser’s built-in Speech Synthesis API. When the user clicks the “Read Text” button, JavaScript collects the webpage text and converts it into spoken audio using the browser speech engine.
